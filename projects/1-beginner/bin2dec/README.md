# Bin2Dec

A binary (base 2) to decimal (base 10) converter.

Source idea: [app-ideas / Bin2Dec](https://github.com/florinpop17/app-ideas/blob/master/Projects/1-Beginner/Bin2Dec-App.md)

## Features

- Live conversion as you type.
- Input validation — only `0` and `1` are accepted, anything else shows an error.
- Supports any number of binary digits (the bonus feature; not limited to 8).

## Constraints honoured

- No arrays are used to store the binary digits.
- Conversion uses a single mathematical function (`Math.pow`).

## Run it

Open `index.html` in any browser — no build step or dependencies.
