# Border Radius Previewer

A small visual tool for rounding the corners of a box and reading off the
generated `border-radius` CSS.

Source idea: [app-ideas / Border Radius Previewer](https://github.com/florinpop17/app-ideas/blob/master/Projects/1-Beginner/Border-Radius-Preview.md)

## Features

- Four sliders — one per corner (top-left, top-right, bottom-right, bottom-left).
- Live preview of the box updating as you drag.
- The matching CSS rule is shown beneath the preview.
- **Copy** button puts the rule on your clipboard (the bonus feature).

## Run it

Open `index.html` in any browser — no build step or dependencies.
